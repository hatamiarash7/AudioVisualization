# Audio Visualization

Audio visualization developed by JavaScript.

![audio](https://user-images.githubusercontent.com/16325641/136423841-e0ad2183-8b37-478f-bd7d-0ce01a2faf21.jpg)

```sh
docker-compose up --build
```
